Task Laundrette

Android Application Development

Group Project of IS3261 Mobile Apps Development for Enterprise

AY2015/2016, SEM I

National University of Singapore

Final Product Preview

Concept Elaboration

Task Laundrette is a 2-person group project product developed according to IS3261's course requirements. Based on the concept of "laundry list", Task Laundrette aims to build up a community for everybody to support each other on small tasks. Every time the user need some help (e.g., pet-sitting, gardening, baking, etc.), he / she can publish a task post in this community. At the same time, if the user finds a task that he / she is interested in, he / she can make an offer to help.
